# sublime 使用记录
1. 失去焦点自动保存
```use
在setting文件中加"save_on_focus_lost": true
结果如
{
	"font_size": 9,
	"ignored_packages":
	[
		"Vintage"
	],
	"save_on_focus_lost": true
}

```
